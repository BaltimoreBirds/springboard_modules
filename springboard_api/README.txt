
-- SUMMARY --

Provides a partial REST API for interacting with webforms via 3rd party code. 


-- REQUIREMENTS --

* Services module
  - REST Server module


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Add a new service via the Services module admin screen.

* Under resources, check webform. This will add index, retrieve and submit resources.
