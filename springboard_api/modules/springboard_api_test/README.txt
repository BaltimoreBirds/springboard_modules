INTRODUCTION
------------

Springboard API test provides a basic testing platform for testing Springboard
connections from a remote site. Springboard API itself includes an internal testing client, this provides one from another Drupal site.

Springboard API class: Contained in the include folder of this module is a class implementation against Springboard API. It can be copied and used in custom PHP systems, using the module as a demonstration of use.

INSTALLATION
------------

1. Copy this directory to your sites/SITENAME/modules directory.

2. Enable the module.
