
-- SUMMARY --
This is a catch-all module to store PCI compliance related code for Springboard. Currently it disables Ubercart's credit card debug mode, which permits storing credit card data in the database.


-- REQUIREMENTS --
None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --
None.
