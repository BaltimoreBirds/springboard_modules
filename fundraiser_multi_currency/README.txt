
-- SUMMARY --

This module adds support for multiple currencies to donation forms. 


-- REQUIREMENTS --

* Fundraiser module.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Currencies are configured via the admin interface: admin/settings/currencies


-- TROUBLESHOOTING --

* Currencies not showing on donation forms:

  Currencies are added when donation forms are created. If you set up a donation form before configuring this module you will need to recreate the donation form to have the latest currency settings applied.
