
-- SUMMARY --

Automatically generate test donations via the Devel module. 

-- REQUIREMENTS --

* Fundraiser module

* Devel Generate module


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

None.

-- Usage --

To generate donations submit the form @ admin/settings/fundraiser/generate
